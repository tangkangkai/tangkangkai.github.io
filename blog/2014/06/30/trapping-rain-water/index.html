Trapping Rain Water

Jun 30th, 2014

Question

For every slot, we find both the highest elevation of its left slots and right slots, the smaller of which will be the local maximum height. Next we need to compare this height, say maxh with the height of the slot, say h, if h is larger, meaning itâ€™s not able to trap water as it itself is the highest. If not, we can trap mh - h amount of water. Here follows the code:

 1    public int trap(int[] A) {
 2        if (A.length == 0) {
 3            return 0;
 4        }
 5        
 6        /* Initialize leftMax and rightMax */
 7        int leftMax = A[0];
 8        int rightMax = 0;
 9        for (int i = 1; i < A.length; i++) {
10            if (A[i] > rightMax) {
11                rightMax = A[i];
12            }
13        }
14        
15        /* Get capacity of every slot */
16        int capacity = 0;
17        for (int i = 1; i < A.length - 1; i++) {
18            int left = 0;
19            int right = 0;
20
21            if (leftMax > A[i]) {
22                left = leftMax;
23            } else {
24                leftMax = A[i];
25            }
26
27            if (rightMax > A[i]) {
28                right = rightMax;
29            } else {
30                int newRightMax = 0;
31                for (int j = i + 1; j < A.length; j++) {
32                    if (A[j] > newRightMax) {
33                        newRightMax = A[j];
34                    }
35                }
36                rightMax = newRightMax;
37            }
38
39            int localMax = left > right ? right : left;
40            capacity += localMax > A[i] ? localMax - A[i] : 0;
41        }
42
43        return capacity;
44    }
