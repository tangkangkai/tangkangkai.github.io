Jun 29th, 2014

Question

This question is an easier version of Remove duplicates in array. Below follows the code:

 1    public int removeElement(int[] A, int elem) {
 2        int offset = 0;
 3    
 4        for (int i = 0; i < A.length; i++) {
 5            if (A[i] == elem) {
 6                offset++;
 7            } else {
 8                A[i - offset] = A[i];
 9            }
10        }
11    
12        return A.length - offset;
13    }
