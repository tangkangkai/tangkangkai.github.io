Next Permutation

Jun 29th, 2014

Question

For this question, in order to get the next larger permutation, we need start from the last element, compare it with the previous element to see if itâ€™s larger(if not, swapping them would make the number smaller). We start from index arr.length -2 to 0 to see which element could be swapped, say we are now at index i, then we use j to traverse from index arr.length -1 to i - 1 to see if arr[i] is smaller than arr[j], if so we can safely swap them(As if we come to index i, the elements from index i + 1 to arr.length -1 must be in decreasing order, otherwise, the swap would happen before i). After we swap, there need two things to be done, first we sort the array after index i, second we return directly. If the previous requirement not met, we simply sort and return.

 1public void nextPermutation(int[] num) {
 2    int lastIndex = num.length - 1;
 3    for (int i = lastIndex - 1; i >= 0; i--) {
 4        for (int j = lastIndex; j > i; j--) {
 5            if (num[i] < num[j]) {
 6                swap(num, i, j);
 7
 8                for (int m = i + 2; m <= num.length - 1; m++) {
 9                    for (int n = m; n > i + 1 && num[n] < num[n - 1]; n--) {
10                        swap(num, n, n - 1);
11                    }
12                }
13                return;
14            }
15        }
16    }
17
18    Arrays.sort(num);
19}
20
21/* Helper Functions */
22public void swap(int[] num, int i, int j) {
23    int temp = num[i];
24    num[i] = num[j];
25    num[j] = temp;
26}
