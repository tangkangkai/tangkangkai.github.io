Relearn Java Day 4 OOP

Jun 29th, 2014

  • member variable:
    • when object created, the corresponding member variables are initialized with default value (as the object is created in heap)
    • the difference between member variable and local variable:
      > 1.member variable is defined in the block of class, can be visited through the class. local variable is defined in the block of function.
      2.member variable is created in heap, while local variable in stack.
      3.member variable exists with class, local variable exists with it’s corresponding blcok(functions).
      4.member variable has default value, local variable doesn’t
  • anonymous object
    • new Car();
    • when member functions only called once, we can use anonymous object
    • can be regarded as function arguments.
